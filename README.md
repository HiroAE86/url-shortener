## Tech Stack

## Getting Started

1. Install dependencies:

```bash
bun install
```

2. Run the local infra

```bash
    docker compose up -d
```

3. Run the app with

```bash
bun run dev
```

## Environment Variables

```env
DATABASE_URL=<postgres_urlk>
PORT=3000
```
